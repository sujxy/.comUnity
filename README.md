# .comUnity
MERN Social Media App for College Campus
## Installation

get started by downloading node_modules into both client & server directories

```bash
  cd client
  cd server
```
```bash
  npm install
```
## Deployment

To deploy this project run (can use node instead of nodemon)

```bash
  cd server
  nodemon index.js
  cd client 
  npm start
```
